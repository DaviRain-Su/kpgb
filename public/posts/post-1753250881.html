静态网站生成器完成！

今天我们成功实现了KPGB的静态网站生成功能。

新增功能

1. 模板引擎

  • 使用Tera作为模板引擎
  • 支持继承和包含
  • 灵活的模板系统

2. 静态网站生成

  • 自动生成HTML文件
  • 响应式设计
  • 美观的CSS样式

3. RSS订阅

  • 标准RSS 2.0格式
  • 自动更新
  • 包含完整文章内容

4. 站点配置

  • site.toml配置文件
  • 自定义标题、描述、作者
  • IPFS网关配置

使用方法

# 初始化配置
cargo run -- init

# 生成网站
cargo run -- generate

# 启动服务器
./serve.sh

技术栈

  • Rust - 主程序
  • Tera - 模板引擎
  • pulldown-cmark - Markdown解析
  • RSS - 订阅源生成
