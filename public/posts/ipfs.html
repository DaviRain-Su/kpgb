IPFS 图片自动上传功能

这是一个测试IPFS图片自动上传功能的文章。

功能说明

当你在文章中使用本地图片时，KPGB会自动：

  1. 检测文章中的本地图片路径
  2. 将图片上传到IPFS
  3. 将本地路径替换为IPFS链接
  4. 保持图片的alt文本不变

支持的图片格式

KPGB支持以下图片格式的自动上传：

  • PNG (.png)
  • JPEG (.jpg, .jpeg)
  • GIF (.gif)
  • WebP (.webp)
  • SVG (.svg)

使用方法

创建新文章时，只需要使用本地图片路径：

cargo run -- new --content my-post.md

编辑文章时也会自动处理新添加的图片：

cargo run -- edit <post-id> --content updated-post.md

这样就能让你的图片永久保存在IPFS上了！
