KPGB Blog Widget Demo

这个页面演示了如何在外部网站中嵌入 KPGB 博客小部件。

基础小部件

最简单的集成方式，显示最新的5篇文章：

<div id="kpgb-widget" data-base-url="http://localhost:8000"></div>
<script src="http://localhost:8000/js/widget.js"></script>

带搜索功能的小部件

启用搜索功能，让用户可以搜索博客内容：

<div class="kpgb-widget" 
     data-base-url="http://localhost:8000" 
     data-search="true"
     data-count="3">
</div>

配置选项

属性 说明 默认值
data-base-url 博客的基础 URL 当前域名
data-count 显示的文章数量 5
data-theme 主题（"light" 或 "dark"） "light"
data-search 启用搜索（"true" 或 "false"） "false"
data-tags 显示标签（"true" 或 "false"） "true"

生产环境部署

部署到 GitHub Pages 后，将 URL 替换为：

<div id="kpgb-widget" data-base-url="https://YOUR_USERNAME.github.io/kpgb"></div>
<script src="https://YOUR_USERNAME.github.io/kpgb/js/widget.js"></script>
